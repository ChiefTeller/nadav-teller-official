# Nadav Teller Official — אתר GitHub Pages

אתר סטטי מהיר בעברית, בסגנון אפליקציה, מוכן להעלאה ל־GitHub Pages.

## איך מעלים
1. פתח את הריפוזיטורי `nadav-teller-official` בגיט־האב.
2. העלה את כל הקבצים והתיקיות מתוך התיקייה הזו אל שורש הריפוזיטורי, לא את קובץ ה־ZIP עצמו.
3. עבור אל Settings → Pages.
4. בחר Deploy from a branch → branch: main → folder: /root.
5. שמור והמתן עד שגיט־האב ייצור את האתר.

## כתובת האתר
https://chiefteller.github.io/nadav-teller-official/

## דברים שכדאי לעדכן
- בקובץ `books.html` יש ספר אחד שסומן: "ספר נוסף באמזון — לעדכון שם וכריכה" עבור ASIN `B0H2HPW4HS`. עדכן אותו כשהשם הסופי ברור.
- כאשר יהיה לוגו, שמור אותו בתיקייה `assets/img/` והחלף את סימן `נט` באזור ה־header.
- הוסף תג אימות של Google Search Console בתוך ה־`head` אם גוגל נותן לך תג.

## SEO
הקבצים כוללים:
- `sitemap.xml`
- `robots.txt`
- canonical URLs
- Open Graph
- Schema JSON-LD לעמוד הספרים ולעמוד תורת המדינה הדואלית
- עמוד עומק מיוחד: `dual-state.html`
